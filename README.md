# Obsidian lab POC

Wants to be: an easy way to experiment with distinct algorithms.

## Installation

Clone this into

/{vault}/.obsidian/plugins

Install dependencies

```
yarn install
```

build the app

```
yarn build
```

This will build a main file. Activate the plugin. 

----

Run the [server](./examples/server.py)
